# 🎬 Studio Ghibli Explorer

A clean and interactive web application that allows users to explore Studio Ghibli films, characters, locations, species, and vehicles using a modern interface powered by the Studio Ghibli API.

---

## ✨ Overview

Studio Ghibli Explorer is a frontend project built with **vanilla JavaScript**, **Bootstrap 5**, and the **Studio Ghibli API**.  
It focuses on delivering a smooth user experience with features like pagination, search, filtering, and detailed movie views.

The goal of this project is to demonstrate real-world frontend development skills such as API consumption, state management, UI rendering, and responsive design.

---

## 🚀 Features

### 🎬 Movie Explorer
- Browse all Studio Ghibli films
- Responsive card layout
- Movie posters and metadata display

### 🔎 Search System
- Live search by movie title
- Instant filtering results

### 🎛️ Filters & Sorting
- Filter by director
- Sort by:
  - Release year (asc/desc)
  - Rotten Tomatoes score
  - Title (A–Z / Z–A)

### 📄 Pagination
- Displays **3 movies per page**
- Dynamic page navigation
- Works with search and filters

### 🎬 Movie Details Page
- Dedicated page per movie (`movie.html?id=...`)
- Full banner display
- Complete metadata:
  - Director
  - Producer
  - Release date
  - Running time
  - Score
- Link to API endpoint

### 🌐 SEO & Sharing Support
- Open Graph meta tags
- Twitter Card support
- Rich link previews
